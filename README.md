<br> 
이 프로그램은 전력전자 시스템의 핵심인 Buck/Boost 컨버터의 초기 설계를 돕고 성능을 분석하는 공학용 소프트웨어입니다. 먼저 설계자로부터 입력 전압, 목표 출력 전압, 인덕턴스, 그리고 스위칭 주파수를 입력받습니다. 프로그램은 입력된 물리량을 바탕으로 전력 변환의 핵심인 듀티비(Duty Cycle)를 계산하고, 인덕터에서 발생하는 전류 리플의 크기를 정밀하게 산출합니다. 계산된 리플 값이 허용 오차 범위 내에 있는지 판정하여 설계 적합성 등급을 매겨주며, 전력 손실 데이터를 기반으로 시스템 효율(%)을 도출합니다. 만약 리플이 너무 크거나 효율이 낮다면 '설계 수정 제안' 메시지를 띄웁니다. 향후에는 다양한 소자 파라미터를 리스트로 관리하고, 최종 설계 리포트를 파일로 출력하는 기능을 추가하여 전공 수업 및 실무에서 활용 가능한 도구로 완성할 계획입니다.
